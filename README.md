# DataBlock : Country Codes

## Install

```bash
composer install datablock/countrycodes
```

## Usage

### Instantiate

```php
$countrycodes = new CountryCodes();
```

### Database

```php
$countrycodes->getDatabase();
```

### Database as jSon

```php
$countrycodes->getJsonDatabase();
```

```php
$countrycodes->getCountryName();
```

```php
$countrycodes->getIso2();
```
